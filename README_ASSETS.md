# 🎨 Руководство по кастомизации оформления квеста-олонхо

В этом приложении реализована гибкая система изменения тем, иконок, кастомных задних фонов и изображений священных столбов **Сэргэ**.

Все настройки берутся из конфигурационного файла `/public/data/custom_config.json`, а медиа-файлы считываются из папки `/public/images/`.

---

## 1. Настройка заднего фона (Карты Тропинки)

Для установки собственного фонового изображения:
1. Подготовьте изображение в формате JPG или PNG (рекомендуется вертикальное изображение с высоким разрешением, например, солнечная поляна, летний луг или тайга).
2. Переименуйте его в **`quest_bg.jpg`** (или измените имя файла в конфигурационном файле).
3. Поместите этот файл в папку `/public/images/` по следующему пути:
   ```text
   /public/images/quest_bg.jpg
   ```
4. Обновите страницу приложения. Фоновое изображение тропы автоматически подгрузится!

---

## 2. Кастомизация через `custom_config.json`

Вы можете открыть файл `/public/data/custom_config.json` и настроить следующие параметры:

```json
{
  "questName": "Якутский олонхо «Ньургун Стремительный»",
  "backgrounds": {
    "mainBackgroundUrl": "/images/quest_bg.jpg",
    "mainBackgroundFallback": "linear-gradient(to bottom, #7dd3fc, #bae6fd, #f0fdf4, #dcfce7)",
    "sargeWoodColorLight": "#fde68a",
    "sargeWoodColorMedium": "#f59e0b",
    "sargeWoodColorDark": "#b45309",
    "trailPrimaryColor": "#c5a880",
    "trailDottedColor": "#78350f"
  },
  "customImages": {
    "sargeActiveImage": "",
    "sargeCompletedImage": "",
    "sargeLockedImage": "",
    "generalSargeImage": ""
  },
  "customIcons": {
    "active": "horse",
    "completed": "check",
    "locked": "lock"
  }
}
```

### Разбор ключевых полей:

1. **`mainBackgroundUrl`**: Относительный путь к фоновой картинке. Если этот файл существует, он перекроет стандартный красивый градиент.
2. **`mainBackgroundFallback`**: Резервный CSS-градиент, если картинка не задана.
3. **Цвета Сэргэ (`sargeWoodColor...`)**: Коды цветов для придания дереву Сэргэ кастомного оттенка (светлое дерево, мореный дуб, сосна и т.д.).
4. **`trailPrimaryColor`** и **`trailDottedColor`**: Цвета извилистой тропинки (по умолчанию песчано-земляная тропинка с темно-коричневыми следами копыт коня).
5. **`customImages`**: Если вы хотите полностью заменить векторные Сэргэ на собственные картинки или фотографии, просто укажите пути к ним в этих полях (например, `/images/serge_active.png`). При наличии путей приложение будет рендерить ваши фотографии/картинки вместо 3D-векторных Сэргэ!
6. **`customIcons`**: Выбор центральных значков.

---

## 3. Смена Иконок и Изображений Сэргэ

Если вы загрузили собственные фотографии реальных Сэргэ, вы можете настроить их отображение в зависимости от статуса задания:

- **Активное задание**: укажите путь в поле `"sargeActiveImage"` (например, `"/images/my_active_serge.png"`).
- **Пройденное задание**: укажите путь в поле `"sargeCompletedImage"`.
- **Заблокированное задание**: укажите путь в поле `"sargeLockedImage"`.

Если эти поля пустые, приложение использует встроенный интерактивный трехмерный векторный Сэргэ из светлого свежего дерева с развивающимися священными лентами салалама!
