# История изменений (CHANGELOG)

Все заметные изменения в проекте **AI Quest Platform** документируются в этом файле.

---

## [1.0.0] — 2026-07-09
### Добавлено
*   **Архитектурное ядро**: Реализована Clean Architecture с разделением на изолированные доменные модули (`src/packages`).
*   **Quest Engine**: Реализован конечный автомат (State Machine) игрового прогресса, система расчета уровней, опыта (XP), выдачи игровых званий, достижений и инвентаря игрока. Добавлена поддержка чекпоинтов и отката состояния (Rollback).
*   **AI Engine**: Разработан контекстный генератор на базе SDK `@google/genai` (модель `gemini-2.5-flash`), адаптивный Prompt-менеджер с защитой от инъекций и обхода правил, а также система автоматического переключения на локальный NLP-обработчик правил в случае превышения лимитов внешнего API.
*   **Vision Engine**: Разработан конвейер компьютерного зрения с поддержкой распознавания текста (OCR), верификации схожести изображений с эталонами, декодирования QR-кодов и оптимизацией производительности с помощью кэширования результатов.
*   **Realtime Engine**: Реализован Socket.IO шлюз с гарантированной доставкой транзакционных событий (ACK/Retry), синхронизацией сессий по `lastEventId` при переподключении и сервисом отслеживания присутствия игроков в реальном времени (Presence Service).
*   **PWA Клиент**: Добавлен адаптивный интерфейс с поддержкой оффлайн-режима (Service Worker, кэширование статики, IndexedDB для оффлайн-сценариев), доступом к системной камере и GPS.
*   **Панель Администратора**: Создан интерактивный дашборд организатора с live-статистикой онлайна, редактором сценариев, промптов и инструментами ручного зачета шагов игрокам.
*   **Инфраструктурные чертежи**: Созданы конфигурации контейнеризации Docker Multi-stage, Docker Compose, манифесты Kubernetes (HPA, PVC, Ingress), Helm Chart, Nginx и Traefik с поддержкой WebSockets и TLS 1.3. Разработан стек мониторинга (Prometheus, Alerts rules, Tempo distributed tracing).
*   **Резервное копирование**: Написаны автоматизированные скрипты периодического архивирования СУБД Postgres и S3 хранилища MinIO с политиками ротации архивов старше 7 дней.
*   **Тесты и Качество**: Внедрена автоматическая система сквозного интеграционного тестирования всего бэкенда и доменных движков. Настроен строгий контроль типов TypeScript и линтер ESLint.
